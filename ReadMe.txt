The game is designed as a memory game. The game will start with 5 numbers 
and as the level increases, the number to remember will increase. 
As the level increases, the time will decrease and the number size will decrease. 
This game is designed for endless use. 
For this reason, the decrease in font size after a certain level and the 
shortening of the duration are fixed.
Finally, the images and code given in the rar file must be in the same file 
directory in order to run the game.
--------------------------------------------------------------------------------------------
Oyun bir hafıza oyunu olarak tasarlanmıştır. Oyun 5 sayı ile başlayacak 
ve seviye arttıkça hatırlanması gereken hane sayısı artacaktır. 
Seviye arttıkça, oyun zamanı azalacak ve ekranda beliren sayı boyutu azalacaktır. 
Bu oyun sonsuz kullanım için tasarlanmıştır. 
Bu nedenle belirli bir seviyeden sonra yazı tipi boyutundaki küçülme ve süre kısalması 
sabit bir değer alacaktır.
Son olarak, rar dosyasında verilen görseller ve kodun oyunu çalıştırabilmesi için 
aynı dosya dizininde olması gerekir.
